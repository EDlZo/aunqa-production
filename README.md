# ระบบประเมินคุณภาพการศึกษา AUNQA (ESAR)
Website : https://aunqa-esar.vercel.app/
ระบบสนับสนุนการจัดทำรายงานการประเมินตนเอง (Self-Assessment Report) ตามเกณฑ์ AUN-QA สำหรับหลักสูตรระดับอุดมศึกษา

---

## บทบาทผู้ใช้งาน (Roles)

| Role | ชื่อ | สิทธิ์หลัก |
|------|------|------------|
| `system_admin` | System Admin | จัดการระบบทั้งหมด ผู้ใช้ รอบประเมิน ฐานข้อมูล |
| `qa_admin` | QA Admin | จัดการโครงสร้างองค์ประกอบ ดูรายงานและสรุปผล |
| `sar_manager` | SAR Manager | กำหนดเกณฑ์ เป้าหมาย จัดการรอบ ตรวจสอบความครบถ้วน |
| `reporter` | Reporter | กรอกผลการดำเนินงาน อัปโหลดหลักฐาน เขียน SAR |
| `external_evaluator` | External Evaluator | ดูข้อมูลที่ได้รับมอบหมาย ให้คะแนนและข้อเสนอแนะ |
| `executive` | Executive | ดูสรุปผล Dashboard และเปรียบเทียบผลการประเมิน |

---

## การเข้าสู่ระบบ

1. เปิดเว็บไซต์ระบบ
2. คลิกปุ่ม **เข้าสู่ระบบ** มุมขวาบน
3. เลือก **บทบาท (Role)** ที่ต้องการเข้าใช้งาน
4. กรอก **อีเมล** และ **รหัสผ่าน**
5. คลิก **เข้าสู่ระบบ**

> หมายเหตุ: ผู้ใช้ 1 คนสามารถมีได้เพียง 1 Role เท่านั้น หากต้องการเพิ่มผู้ใช้ให้ติดต่อ System Admin

---

## การใช้งานตามบทบาท

### System Admin

เมนูที่เข้าถึงได้:
- **จัดการรอบประเมิน** — เปิด/ปิดรอบการประเมิน กำหนดปีการศึกษา
- **จัดการผู้ใช้** — เพิ่ม แก้ไข ลบผู้ใช้งานในระบบ
- **จัดการระบบ** — ตั้งค่าระดับการศึกษา คณะ และสาขาวิชา
- **จัดการฐานข้อมูล** — นำเข้า/ส่งออกข้อมูล จัดการ Master Indicators
- **จัดการโปรแกรม** — เพิ่ม/แก้ไขโปรแกรมการศึกษา

ขั้นตอนเริ่มต้นใช้งานระบบ:
1. เข้าเมนู **จัดการระบบ** → เพิ่มระดับการศึกษา คณะ และสาขา
2. เข้าเมนู **จัดการผู้ใช้** → สร้างบัญชีผู้ใช้แต่ละ Role
3. เข้าเมนู **จัดการรอบประเมิน** → สร้างรอบและกด **เปิดใช้งาน**

---

### QA Admin / SAR Manager

เมนูที่เข้าถึงได้:
- **จัดการองค์ประกอบคุณภาพ** — กำหนดองค์ประกอบและตัวบ่งชี้ของแต่ละสาขา
- **ตารางประเมิน** — ดูภาพรวมการกรอกข้อมูลของทุกตัวบ่งชี้
- **รายงาน** — ออกรายงาน SAR และ PDF
- **สรุปผล** — ดูผลการประเมินรวม

ขั้นตอนการจัดการองค์ประกอบ:
1. เลือกเมนู **จัดการองค์ประกอบคุณภาพ**
2. เลือกสาขาวิชาที่ต้องการจัดการ
3. เพิ่ม **องค์ประกอบคุณภาพ** (Quality Component)
4. เพิ่ม **ตัวบ่งชี้** (Indicator) ในแต่ละองค์ประกอบ

---

### Reporter

เมนูที่เข้าถึงได้:
- **กรอกผลการประเมิน** — กรอกผลการดำเนินงานและอัปโหลดหลักฐาน
- **ตารางประเมิน** — ดูสถานะการกรอกข้อมูลของตนเอง

ขั้นตอนการกรอกข้อมูล:
1. เลือกเมนู **กรอกผลการประเมิน**
2. เลือกสาขาวิชาและตัวบ่งชี้ที่รับผิดชอบ
3. กรอก **ผลการดำเนินงาน** และ **คะแนน**
4. อัปโหลด **ไฟล์หลักฐาน** (PDF หรือรูปภาพ ขนาดไม่เกิน 20MB)
5. กด **บันทึก**

> ข้อมูลที่ได้รับการอนุมัติแล้วจะไม่สามารถแก้ไขได้

---

### External Evaluator

เมนูที่เข้าถึงได้:
- **ประเมินโดยคณะกรรมการ** — ให้คะแนน จุดแข็ง และข้อเสนอแนะ

ขั้นตอน:
1. เลือกเมนู **ประเมินโดยคณะกรรมการ**
2. เลือกสาขาและตัวบ่งชี้ที่ได้รับมอบหมาย
3. กรอก **คะแนน**, **จุดแข็ง**, **จุดที่ควรพัฒนา** และ **แผนพัฒนา**
4. กด **บันทึก**

---

### Executive

เมนูที่เข้าถึงได้:
- **Dashboard** — ภาพรวมผลการประเมินทุกสาขา
- **สรุปผล** — ดูผลคะแนนรวมและเปรียบเทียบ

---

## หมายเหตุสำคัญ

- เมนูส่วนใหญ่จะถูกล็อกหากยังไม่มี **รอบประเมินที่เปิดใช้งาน** — ให้ System Admin เปิดรอบก่อน
- ไฟล์หลักฐานที่อัปโหลดรองรับเฉพาะ **PDF และรูปภาพ** เท่านั้น
- Session การเข้าสู่ระบบมีอายุ **7 วัน** หลังจากนั้นต้องเข้าสู่ระบบใหม่
